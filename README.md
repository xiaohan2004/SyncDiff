# SyncDiff

该程序用于 快速对比两个文件夹的内容，支持 MD5 校验，并输出相应结果。通过配置文件指定需要对比的文件夹，支持 忽略特定路径或文件（使用正则规则），并能自动处理 文件夹不存在的情况，避免报错。

输出情况如下：
- 文件在两个文件夹中都存在且相同：不输出（默认忽略相同文件）
- 文件在两个文件夹中都存在但内容不同：输出文件相对路径，各自的绝对路径、MD5 值、文件大小和最后修改时间
- 文件仅在某一文件夹存在：输出文件相对路径和所在文件夹标记
- 文件夹不存在：输出警告，跳过该 section

特点包括：
- 支持 配置文件中任意两个以 folder_ 开头的文件夹进行对比
- 输出中自动去掉 folder_ 前缀，更直观
- 大小写敏感，保留实际文件名大小写
- 可在不同工作区下尝试多个配置文件路径，找到有效配置即可运行

适用于：
- 数据备份校验
- 文件同步检查
- 项目多版本文件比对
